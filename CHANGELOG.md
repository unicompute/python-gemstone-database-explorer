# Changelog

## 1.0.0 - 2026-04-25

- Declared the first packaged explorer release and aligned package metadata with the shipped tool surface.
- Added runtime app-version reporting through `/version` so the UI and external checks can identify the running build.
- Completed the windowed explorer toolset: object inspectors, workspace, Symbol List Browser, debugger, helper windows, layout persistence, and related-window management.
- Completed the bridge-style Class Browser workflow, including helper windows, source/version inspection, structure editing, and transaction controls.
- Added mock and live Playwright UI coverage alongside Python route, object-view, and session tests.
